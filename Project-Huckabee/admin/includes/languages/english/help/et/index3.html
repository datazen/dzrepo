What is this?

Once you have navigated to the correct file to edit you then can edit the language defines. The intial edit screen is show above.

Specific Task Help


   1. Identifying the text to edit.
     a. Note the file name in the browser url box.
     b. Next note the text you want to change.

   2. Finding the correct file:
     a. Go to tools > Define Languages.
     b. Select the directory the define file resides in.
     c. Click on the file name of the file.

Editing Language Defines


   1. Type in the new text
   2. Click on save next to the input box you just type in
   3. Double check your changes by viewing them in the cart.
   4. If they did not work or are incorrect, you can restore the previous text if change again in the editor
  Note: you can only sabe changes for one input box at a time.

Restoring Language Defines


   1. Once you have saved the define your changes you can double check that it is correct.
   2. If it is not then you can click on the restore button to revert back to the previous langauge define.
